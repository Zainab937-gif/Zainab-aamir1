The final stage of our Thyroid Cancer Recurrence Prediction Project involved converting our machine learning model into a usable web application. This was an exciting experience because it allowed us to combine database concepts, machine learning, and web development into a single system.

To connect our Random Forest model with the user interface, we used FastAPI as the backend framework. FastAPI handled requests from users and generated predictions based on the information entered into the system. This helped us understand how machine learning models can be deployed in practical applications.

For the frontend, we created pages using **index.html**, **style.css**, and **script.js**. The HTML file provided the structure of the website, CSS improved the appearance and layout, while JavaScript added interactivity. By integrating these technologies with FastAPI, we successfully created a working prediction system.

The most memorable part was presenting our project at the exhibition. Faculty members, teachers, and students visited our stall and asked questions about the database design, Random Forest algorithm, FastAPI integration, and project implementation. Although I was nervous initially, I gained confidence while explaining our work.

Dr. Bilal Ahmad's guidance played a major role throughout the project. His focus on practical learning encouraged us to build a complete solution rather than just a machine learning model. I also followed updates on his <a href="https://www.facebook.com/Dr.BilalAhm">Facebook page</a>, where he shares information about research and student projects.

This project taught me valuable lessons about teamwork, databases, machine learning, and web development. It was one of the most rewarding experiences of my academic journey and inspired me to continue exploring AI-based solutions in the future.
