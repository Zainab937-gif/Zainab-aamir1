The Database Systems course introduced me to many new technologies, and one of the most interesting experiences was learning how to create a website using WordPress. Since this was my first Database lab of the semester, I was excited to learn something different from traditional programming exercises.

In our first practical session, Dr. Bilal Ahmad guided us through the process of setting up a local web development environment. We started by downloading and installing the WAMP Server, which allowed us to run web applications on our computers. Although the installation process seemed complicated at first, the step-by-step instructions made it easier to understand.

After configuring WAMP Server, we learned how to install and connect WordPress with a database. This was my first experience working with a Content Management System (CMS). I was surprised to see how quickly a website could be created without writing large amounts of code. At the same time, I learned that databases play an important role in storing website content, user information, and other important data.

Another important topic introduced during the lab was MySQL. Dr. Bilal Ahmad explained how MySQL is used to create and manage databases that support web applications. Understanding the relationship between WordPress and MySQL helped me realize how websites function behind the scenes. It was interesting to see how data entered on a website is actually stored and managed within a database.

Beyond classroom teaching, Dr. Bilal Ahmad also encourages students to build a professional digital presence. Following his advice, I explored his <a href="https://www.linkedin.com/in/drbilalphd/">LinkedIn profile</a> to learn more about his work in Artificial Intelligence, Machine Learning, and education. It motivated me to think about creating my own technical portfolio in the future.

This lab session was a valuable learning experience because it introduced me to website development, database management, and modern web technologies. It also strengthened my understanding of how software applications and databases work together. Overall, it was an excellent start to my Database Systems journey.
